# 🐾 Superfarmer

Gra planszowa Superfarmer – wersja webowa (React + Express + Redis)

## Jak uruchomić

1. Uruchom serwer Redis:
   ```bash
   redis-server
   ```
2. Uruchom backend (w katalogu `superfarmer-backend`):
   ```bash
   node index.js
   ```
3. Uruchom frontend (w katalogu `superfarmer`):
   ```bash
   npm start
   ```
